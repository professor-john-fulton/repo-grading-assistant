# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation respectful, inclusive, and
harassment-free.

## Standards

Examples of positive behavior: - respectful language - constructive
feedback - empathy

Unacceptable behavior: - harassment - discrimination - trolling -
personal attacks
